# Game Process

1. Start Game
2. Create Context
3. Start Daily Scrum
   1. Create 3 Agenda of Company
4. Update Context
5. Start Work
   1. Create 3 Task of Player
6. Update Context
7. Create Overtime or Rest Task
   1. Create 1 Overtime or Rest Task of Player
8. Update Context
9. if Max turn reached,
   1.  End Game
      1. Create Result of Company
      2. Create Result of Player
